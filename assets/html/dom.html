Manipuler le DOM en JavaScript

Un guide sur la façon de modifier le Document Object Model (DOM) pour rendre les pages web interactives.

Le DOM est une représentation en mémoire de la structure d'un document HTML. Il permet de manipuler les éléments de la page à l'aide de JavaScript.

Voici un exemple de code JavaScript pour manipuler le DOM :

document.addEventListener('DOMContentLoaded', function() {
    var element = document.createElement('p');
    element.textContent = 'Paragraphe ajouté dynamiquement';
    document.body.appendChild(element);
});
