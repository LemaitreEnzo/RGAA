Introduction au JavaScript

Comprenez les bases du JavaScript et apprenez à rendre vos pages interactives.

Le JavaScript est un langage de programmation utilisé pour créer des pages web interactives. Il permet de manipuler le DOM (Document Object Model), de gérer les événements, de valider les formulaires, et bien plus encore.

Voici un exemple de code JavaScript de base :

document.addEventListener('DOMContentLoaded', function() {
        document.querySelector('button').addEventListener('click', function() {
            alert('Bouton cliqué !');
        });
    });
