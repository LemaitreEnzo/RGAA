Tutoriel HTML de base

Découvrez les fondamentaux du HTML et créez des pages web avec des balises sémantiques.

Le HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il permet de structurer le contenu et de définir les éléments de la page tels que les titres, les paragraphes, les images, les liens, etc.

Voici un exemple de code HTML de base :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
</head>
<body>
    <h1>Titre principal</h1>
    <p>Ceci est un paragraphe.</p>
</body>
</html>
