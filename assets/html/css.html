CSS pour les débutants

Apprenez à styliser vos pages web avec des sélecteurs CSS, des couleurs, et bien plus encore.

Le CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire la présentation des documents HTML. Il permet de contrôler l'apparence des éléments de la page, tels que les couleurs, les polices, les marges, les bordures, etc.

Voici un exemple de code CSS de base :

body {
        font-family: Arial, sans-serif;
        background-color: #f4f4f4;
        color: #333;
    }
    
    h1 {
        color: #e10098;
    }
