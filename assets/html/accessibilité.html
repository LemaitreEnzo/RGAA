Accessibilité

Chez Tech'n'Tips by La Manu, nous nous engageons à rendre notre site web accessible à tous, y compris aux personnes en situation de handicap. Nous suivons les directives du RGAA (Référentiel Général d'Amélioration de l'Accessibilité) pour garantir une expérience utilisateur optimale pour tous.

Principes de base de l'accessibilité

  • Utilisation de balises HTML sémantiques pour structurer le contenu.
  • Ajout de textes alternatifs pour les images et les éléments multimédias.
  • Contraste suffisant entre le texte et l'arrière-plan pour une meilleure lisibilité.
  • Navigation au clavier pour permettre l'accès à toutes les fonctionnalités sans utiliser de souris.
  • Utilisation de rôles ARIA pour améliorer l'accessibilité des éléments interactifs.

Nos engagements

Nous nous engageons à :

  • Effectuer des tests réguliers pour vérifier la conformité de notre site avec les normes d'accessibilité.
  • Apporter des améliorations continues pour garantir une accessibilité optimale.
  • Prendre en compte les retours des utilisateurs pour améliorer l'expérience utilisateur.
